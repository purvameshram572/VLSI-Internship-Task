#*VLSI Internship – Task 1*
Digital Logic Gate Implementation, Half Adder and Full Adder Verification

#📌 *Project Overview*

This repository contains the work completed during Week 1 of the VLSI Engineering Internship. The objective of this task was to understand the fundamentals of digital logic design, implement basic logic gates using Logisim Evolution, and verify the functionality of Half Adder and Full Adder combinational circuits.

---

#*Objectives*

- Understand the fundamentals of VLSI design.
- Study the VLSI design flow.
- Implement basic digital logic gates.
- Verify gate functionality using truth tables.
- Design and simulate a Half Adder circuit.
- Design and simulate a Full Adder circuit.
- Analyze signal flow and output behavior through simulation.

---
# 🛠️ Tools Used

* **Primary Schematic Simulator:** `Logisim` (Recommended offline digital logic simulator)
* *Alternative environments:* CircuitVerse, Falstad, or Digital (by H. Neemann)
* 🔧 To view and test the circuits, please download the `.circ` file and open it using **Logisim**.
