# VLSI-Internship-Task 1
Technical Reports and Simulations for VLSI Internship at Mainscrafts Technology.

## Project Overview
This repository contains the tasks completed during the VLSI design internship.

### Task 1: Introduction to VLSI Design Flow and Basic Digital Logic Implementation
* Study of the 9-stage semiconductor design flow.
* Verification of basic digital logic gates.
* Implementation and simulation of a functional Half Adder circuit using digital logic tools.
* Identification and correction of circuit connection errors (fixing wiring tool issues).
